# Toolbox

A versatile and lightweight Android utility app that brings essential everyday tools into a single, clean interface. Built with simplicity and user privacy in mind.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Android](https://img.shields.io/badge/Platform-Android-green.svg)](https://www.android.com)

## Key Features

Toolbox provides a seamless experience for common tasks without the clutter of multiple separate apps.

### Included Tools

- **Calculator**: A straightforward calculator for quick arithmetic and mathematical operations.
- **Notepad**: Create and manage notes easily with built-in storage.
- **Stopwatch**: Track elapsed time with lap support and a foreground service for reliability.
- **Timer**: Set and manage multiple timers for your daily activities.
- **Random Number Generator**: Generate random numbers quickly for games or decision-making.

## Why Toolbox?

- **No Ads**: Enjoy a clean experience without any interruptions.
- **Privacy First**: We do not collect, store, or share any user data.
- **Lightweight**: Minimal resource usage and fast performance.
- **Smart Navigation**: Uses a `BottomNavigationView` and remembers your most used tools to prioritize them.
- **Auto-Updates**: Integrated version checking to keep you up to date with the latest features.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/iasonasTan/Toolbox.git
    ```
2.  Open the project in **Android Studio**.
3.  Build and run on your Android device or emulator (Min SDK 35).
